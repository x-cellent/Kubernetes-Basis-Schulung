1. Install helm binary
2. Helm Chart aus den derzeitigen Objekten erstellen
   - Webserver deployment + (sidecar) + (init)
   - Service
   - ServiceAccount
   - ConfigMap
   - RoleBinding
3. Installieren Sie nun dieses Helm Chart in das Cluster in einen neuen Namespace
